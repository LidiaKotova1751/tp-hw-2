# Система управления рецептами

Это проект системы управления рецептами на языке Python, разработанный в рамках домашнего задания курса "Технологии программирования".

Приложение позволяет:

- Создавать ингредиенты и рецепты блюд (включая классификацию типа "диетическое")
- Масштабировать порции
- Автоматически генерировать и объединять списки покупок, суммируя одинаковые ингредиенты из разных рецептов.

## Установка

Для развертывания проекта на локальном компьютере выполните следующие шаги:

1. Склонируйте репозиторий:

```bash
git clone https://github.com/LidiaKotova1751/tp-hw-2
cd tp-hw-2
```
2. Установите необходимые зависимости:

```bash
pip install -r requirements.txt
```

при необходимости используйте эту команду:

```bash
python -m pip install -r requirements.txt
```

или 

```bash
py -m pip install -r requirements.txt
```

## Использование

### Запуск кода
Основные классы (`Ingredient`, `Recipe`, `DietaryRecipe`, `ShoppingList`) можно импортировать и использовать в любом Python-скрипте:

```python
# Пример использования
from Classes import Ingredient, Recipe, ShoppingList

# Создаём рецепт
flour = Ingredient("Мука", 200, "г")
milk = Ingredient("Молоко", 100, "мл")
pancakes = Recipe("Блины", [flour, milk])

# Добавляем в список покупок на 3 порции
shop_list = ShoppingList()
shop_list.add_recipe(pancakes, portions=3)

# Получаем итоговый список
print(shop_list.get_list())
```

### Запуск тестов
Проект покрыт unit-тестами с использованием библиотеки `pytest`. Для запуска всех тестов выполните в корневой директории проекта команду:

```bash
pytest
```

Для вывода подробной информации о прохождении тестов используйте флаг `-v`:

```bash
pytest -v
```

при необходимости используйте эти команды:

```bash
python -m pytest
```

или

```bash
py -m pytest
```

и 

```bash
python -m pytest -v
```

или

```bash
py -m pytest -v
```
